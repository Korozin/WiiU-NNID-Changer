# WiiU-NNID-Changer

Just decided to make a GUI for one of my oldest projects.

## Dependencies:

-   PyQt5
-   Classes folder (obviously)

that's it.

## To run:

Run the command:
```cmd
NNIDChanger.py
```
Make sure you have the `Classes` folder as it has dependencies in it.
